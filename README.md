# DataViz 数据可视化工具

一个基于 Streamlit 构建的轻量级数据可视化 Web 应用，支持上传 Excel/CSV 文件，快速生成柱状图、折线图、饼图、环形图、雷达图等多种图表，并针对大数据量进行了优化，支持中文导出。

## ✨ 功能特性

- 📁 **多格式支持**：上传 `.xlsx`、`.xls`、`.csv` 文件，自动识别数值列和类别列。
- 📊 **丰富的图表类型**：柱状图、折线图、饼图、环形图、雷达图，满足不同分析场景。
- ⚡ **大数据处理**：当数据超过 5000 行时，自动启用大数据处理模式，提供随机采样、分组聚合（求和/平均）、截断等选项，确保图表渲染流畅。
- 🎨 **样式自定义**：图表标题、颜色主题、图例、网格线开关，轻松调整图表外观。
- 🌏 **中文字体支持**：集成 Noto 中文字体，导出图片时中文正常显示，避免方框乱码。
- 💾 **一键导出**：图表可导出为 PNG / JPG / SVG 图片，支持自定义尺寸和背景色（白色/透明）。
- 🖥️ **交互式界面**：实时数据预览，动态调整图表参数，所见即所得。

## 🚀 快速开始

### 1. 环境要求
- Python 3.8 或更高版本
- 推荐使用虚拟环境（venv 或 conda）

### 2. 安装依赖
克隆或下载本项目后，在终端执行：

```bash
pip install -r requirements.txt
或手动安装：

bash
pip install streamlit pandas plotly openpyxl kaleido
3. 运行应用
bash
streamlit run app.py
或使用 Python 模块方式：

bash
python -m streamlit run app.py
浏览器将自动打开 http://localhost:8501，即可使用工具。

📊 使用说明
上传数据：点击左侧“浏览文件”或拖拽文件到指定区域，支持 Excel 和 CSV 格式。

选择 X/Y 轴：从下拉菜单中选择用于类别轴和数值轴的列。

选择图表类型：在侧边栏选择柱状图、折线图、饼图、环形图或雷达图。

大数据处理（可选）：若数据超过 5000 行，会显示处理选项，可选择随机采样、分组聚合或截断。

调整样式：设置标题、颜色主题、图例、网格线等。

生成图表：点击“生成图表”按钮，图表将显示在主区域。

导出图片：设置导出格式、尺寸和背景色后，点击“导出图表”即可下载。

❗ 常见问题
Q: 导出图片时中文显示为方框？
A: 请确保部署环境中已安装中文字体。在 Streamlit Cloud 中，需在 packages.txt 中添加 fonts-noto-cjk，并在代码中设置字体族（项目已集成）。本地运行时，系统需有中文字体（如 Windows 的“微软雅黑”）。

Q: 大数据量时图表生成卡顿？
A: 项目已集成大数据处理功能。当数据超过 5000 行时，会自动显示处理选项，选择“随机采样”或“分组聚合”可显著减少数据量，提升渲染速度。

Q: 雷达图显示不全？
A: 雷达图默认最多显示 50 个点，超出时会自动截断并给出提示。建议先对数据进行聚合或筛选。

Q: 如何修改图表颜色主题？
A: 在侧边栏“样式设置”中选择不同的颜色主题，支持蓝色、绿色、紫色、红色、橙色和多彩色盘。
